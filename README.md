# Podcast Research Automation

Automated research system for **The Dime Podcast** that delivers high-quality guest research documents 3 days before each recording.

---

## What This Does

**Automatically:**
1. Monitors Trello board daily at 9:00 AM EST
2. Detects recordings scheduled exactly 3 days out
3. Extracts guest information from Trello cards | Name | Title | Company| 
4. Researches guest via Claude AI (podcasts, content, patterns, belief systems) Using Pre built prompt 
5. Generates research document per specification
6. Emails completed research document


## What You'll Receive

Each research document contains:

**Section 1: Core Insight Map**
**Section 2: Tension Opener Questions**
**Section 3: Escalation Paths**

**Section 4: SEO Layer**
